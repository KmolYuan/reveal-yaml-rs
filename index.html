The index of fragment is the order of the animation.

The blank / ~ / null represents normal class="framgment" effect.

hstack:
  - doc: AAA
    math: f(x)
    fragment: [ { doc: ~, math: ~ } ]  # one action
  - doc: BBB
    math: g(y)
    fragment: [ doc: ~, math: ~ ]  # two action
  - doc: CCC
    math: h(z)
    fragment: [ math: ~ ]  # one element

Third-party Plugins

The third-party plugins can be loaded from metadata.

plugin:
  RevealNotes:
    - plugin/notes/notes.js

Will be converted to:

<script src="/plugin/notes/notes.js"></script>

Reveal.initialize({
  plugins: [..., RevealNotes],
  ...
});
